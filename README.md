# 🧠 Atividade de Algoritmos - Passos Lógicos

Este repositório contém a resolução de uma atividade de algoritmos, onde foram desenvolvidos passos lógicos para tarefas do dia a dia.

---

## 📋 Atividades
- Preparar um brigadeiro  
- Rotina de acordar e ir ao trabalho/escola  
- Preparar um café  
- Preparar um x-burguer  

---

## 🎯 Objetivo
Desenvolver o raciocínio lógico através da criação de algoritmos simples, com no mínimo 5 e no máximo 10 passos.

---

## 🚀 Aprendizados
Este projeto também foi utilizado como objeto de estudo para:
- Relembrar conceitos de HTML e CSS  
- Praticar estilização com Tailwind CSS  
- Exercitar a leitura de documentação
